# Element-UI要覆盖默认样式怎么做

由于 element-ui 的样式我们是在全局引入的，所以你想在某个view里面覆盖它的样式就不能加 scoped，但你又想只覆盖这个页面的 element 样式，你就可在它的父级加一个 class，以用命名空间来解决问题

/* 你的命名空间 */
.article-page {
  /* element-ui 元素 */
  .el-tag {
    margin-right: 0px;
  }
}

# Echart使用的是svg还是canvas

Canvas：更适合绘制图形元素数量非常大的图表，比如热力图、地理坐标系，也利于实现某些视觉效果。 Svg：内存占有更低（适合移动端）、渲染性能高、并且用户使用浏览器内置的缩放功能不会模糊。

  • 在软硬件环境较好，数据量不大的场景下，都可以使用两种
  • 在创建很多实例，且浏览器易崩溃的时候，可以使用Svg渲染器进行改善。
  • 数据量很大、较多交互时，可以使用Canvas渲染器。
